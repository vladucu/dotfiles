## Usage

[*chezmoi*](https://www.chezmoi.io/) is used to bootstrap dotfiles.

* One-line binary install
    ```sh
    sh -c "$(curl -fsLS chezmoi.io/get)" -- init --apply vladucu
    ```

    <summary>Notes</summary>

### Import GPG keys
